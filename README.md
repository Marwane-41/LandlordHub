Property Management System  | Système de Gestion Immobilière

This is a full-stack property management application that allows administrators to manage tenants and payments efficiently.
The system provides CRUD operations for:

Tenants
Payments

--> The goal of this project is to practice full-stack development using modern web technologies and to understand state management, API's, and UI practice. 

Tech Stack  ( MERN ) 
- Frontend : 
    React
    | Tailwind CSS / DaisyUI

- Backend : 
    Node.js
    | Express.js
    | MongoDB ( My Database )
